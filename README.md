Google Play Store Apps Analysis
Explored tens of thousands of Play Store app listings with Python (pandas) to uncover download trends, category leaders, monetization, and market opportunities.

Skills Demonstrated


Data cleaning and exploratory analysis on large-scale app data.

Grouped and aggregated app metrics: downloads, ratings, engagement per day.

Identified top app categories, download expectations, and competitive leaders.

Evaluated monetization strategies (in-app purchases, ads, pricing).

Mapped category-by-category business potential.

Tools Used


Python (pandas)

Jupyter Notebook

Project Highlights


Ranked app categories by downloads and engagement.

Found top apps in every genre and assessed daily downloads.

Analyzed links between monetization approach and user engagement.
